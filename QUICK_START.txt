🚀 INSTALLATION (5 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Extract the ZIP
────────────────────────────────────────────────────────────────────
$ unzip ilham-project-complete.zip
$ cd ilham-project


STEP 2: Install Dependencies
────────────────────────────────────────────────────────────────────
$ npm install

⏱️  This takes 2-3 minutes


STEP 3: Configure Environment Variables
────────────────────────────────────────────────────────────────────
Edit the .env file with your Supabase credentials:

NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

📝 Get these from: Supabase Dashboard → Settings → API


STEP 4: Apply Database Migrations
────────────────────────────────────────────────────────────────────
Go to Supabase Dashboard → SQL Editor

Copy and paste each migration file in order:
1. supabase/migrations/20260309034059_create_initial_schema.sql
2. supabase/migrations/20260310005448_add_interactions_and_features.sql
3. supabase/migrations/20260311002126_add_marketplace_and_admin_features.sql
4. supabase/migrations/20260311132849_add_additional_phase2_features.sql
5. supabase/migrations/20260311145410_add_platform_news_table.sql

Click "Run" for each one.


STEP 5: Start Development Server
────────────────────────────────────────────────────────────────────
$ npm run dev

✅ Open http://localhost:3000
